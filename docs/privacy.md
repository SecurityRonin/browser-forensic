# Privacy Policy

*Last updated: 2026-05-05*

## Summary

browser-forensic is a local command-line tool and library. It does not collect, transmit, or store any personal data on remote servers.

## Data Access

browser-forensic reads only the files you supply on the command line. All processing happens in memory on your local machine. Nothing is uploaded anywhere.

## Telemetry

browser-forensic has **no telemetry**. It makes no network requests of any kind.

## Open Source

browser-forensic is open source (Apache-2.0). You can audit every line of code at [github.com/SecurityRonin/browser-forensic](https://github.com/SecurityRonin/browser-forensic).
